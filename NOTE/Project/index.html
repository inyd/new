<프로젝트 설명>

건강과 관련된 제품/정보들을 사용자에게 제공 및 추천하는 홈페이지이다. 사용자가 선택한 식생활, 내면, 관련 비디오, 운동에 관한 세부적인 메뉴들에 관하여 소개하여 건강한 삶의 영위를 돕는다.

-기능 소개-

Home을 눌렀을 때 처음의 화면에는 지도 api를 활용해 회사의 위치를 나타내는 새 창으로 넘어간다.

Login과 회원가입을 누를 시 웹 스토리지로 회원의 정보가 담긴 alert를 띄운다.

사용자가 로그인 시 id와 pw를 입력했을 때 해당 ID는 존재하지만 pw가 일치하지 않으면 경고 메시지를 표현하고, 해당 id가 존재하지 않을 시 확인 버튼을 누르면 회원가입 화면으로 이동할 수 있게 한다.
